人事管理项目(前端分支)
============
项目原作者[江南一点雨](https://github.com/lenve/vhr)
----------
`项目二次开发(技术迭代|功能更新)`
-----------
### 项目介绍
微人事是一个前后端分离的人力资源管理系统，项目采用 SpringBoot+Vue 开发，项目加入常见的企业级应用所涉及到的技术点，例如 Redis、RabbitMQ 等。

#### 前端技术栈
1. Vue
2. ElementUI
3. axios
4. vue-router
5. Vuex
6. WebSocket
7. vue-cli4
8. ...
